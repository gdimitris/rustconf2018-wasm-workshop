Wasm Workshop: BoosterConf 2018

This page is located at http://steveklabnik.github.io/booster2018

Overall agenda:

The workshop is structured bottom-up; we'll start by doing everything by hand, and end up showing how to use nice tools at the end. That way, you know how the tools work!

  1. Steve will give an overview of WASM; we'll write some wasm code by hand, and integrate it into a web page
  2. Ashley will show you how to turn Rust into WASM, a more practical way of using it
